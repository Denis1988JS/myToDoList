Проект - mytodoslist

Схема приложения todos

1.Необходимые веб - страницы: (список ниже)
 - главная (index.html) - 
	(Шапка + nav  - логотип (придумать), ссылки на страницы регистрация и авторизация, востановление пароля, страница описание проекта)
	(Главный контент 
		- (форма для добавления задачи)
		- (блок отображение задач у пользователя - группировка по дням)
	)



2.Необходимые модели - 
	модель Задача:- ГОТОВО
		- id
		- title - (оглавление + текст 100)
		- content - (описание + текст 500)
		- photo - (фотка - не обязательно)
		- статус - (выполнен/не выполнен - bool значение ) => 
		- дата создания (дата авто)
		- срок выполнения (выбор пользователем - дата)
		- внешний на пользователя (User, с удалением )

	модель Пользователя - 
		- Логин
		- first name
		- last name
		- email

3.Главный блок - отображение задачи
	3.1. Все задачи пользователя - сортировка по дате создания
	3.2. Кнопка - выполнение задачи - меняет статус
	3.3. Кнопка - удаление задачи - удаляет одну задачу
